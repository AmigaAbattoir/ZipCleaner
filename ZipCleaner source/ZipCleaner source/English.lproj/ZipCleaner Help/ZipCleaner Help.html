ZipCleaner Help

homepage.mac.com/roger_jolly/software/

ZipCleaner is a program that allows you to remove .DS_Store files and other resource information from zip archives created by the Finder.

About ZipCleaner

When you create a zip archive in the Finder, .DS_Store files and other resource information can be included. Although these are useful on a Mac, and will be invisible to the user, they show up when you unzip the archive on other operation systems, e.g. MS Windows. There these files are just meaningless garbage. ZipCleaner lets you remove them from the zip archives the Finder creates.

By default, you can remove all resource information by simply dropping the archive on ZipCleaner. As removing resource forks can make files, (mostly older applications, but also some AppleScripts), unusable, ZipCleaner will warn you. You can change this, by changing the preferences.
Sometimes you might only want to remove the .DS_Store files. To do that, drop an archive of ZipCleaner while pressing the option key, (or another modifier key, you set in the preferences.)

You should be able to open zip files in ZipCleaner from the contextual menu "Open With". Unless you have told ZipCleaner to report the results of its activities, the application will quit when it is done, if you open it by dropping a file on it. It will remain open, if you open ZipCleaner in another way, e.g. by double clicking on its icon in the Finder.
