.DS_Store

Whenever you change the default view of a folder, the Finder creates a .DS_Store file, (Desktop Services Store). This is an hidden file, that stores information about the folder, e.g. he position of the icons. The file also contains the Spotlight Comments of every file inside the folder.
If you create a zip-archive in the Finder of a folder, it will also include .DS_Store files.

By default, you can remove .DS_Store files by dropping the archive on ZipCleaner while pressing the option key. You can tell ZipCleaner to warn you before it does any removing, by changing the preferences.
